Daily Planner

Overview

Daily Planner is a simple yet efficient task management application built using React. It allows users to add tasks with a custom date selection feature. If no date is specified, the task will be assigned to today's date by default. The application ensures that duplicate tasks are not added and provides a responsive interface for a seamless user experience.

Features

Add Tasks with Custom Date: Assign a specific date to a task using the built-in calendar.

Auto Assign Today's Date: If no date is selected, the task will be scheduled for today.

Duplicate Task Prevention: Prevents users from adding the same task multiple times.

Responsive Design: Ensures usability across different screen sizes and devices.
