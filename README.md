# 📅 Daily Planner

## 📝 Overview  
Daily Planner is a user-friendly task management application built with React, designed to help users organize their daily tasks efficiently. It provides an intuitive interface for managing tasks, ensuring a seamless experience across different devices. Whether planning ahead or keeping track of daily responsibilities, this application makes task management effortless and structured.


## 🚀 Features  
- **Add Tasks with Custom Date**: Assign a specific date to a task using the built-in calendar.  
- **Auto Assign Today's Date**: If no date is selected, the task will be scheduled for today.  
- **Duplicate Task Prevention**: Prevents users from adding the same task multiple times.  
- **Mark Task as Completed**: Users can mark a task as finished to keep track of progress. ✅  
- **Delete Tasks**: Users can remove tasks they no longer need. ❌  
- **Responsive Design**: Ensures usability across different screen sizes and devices.  

## 🛠️ Technologies Used  
- **React.js**  
- **JavaScript**  
- **HTML & CSS**
